Project 4 - "Team Performance Analysis Dashboard"

Commentary / Analysis:

Code uses Matplotlib to create a bar plot comparing the hours played by players in two different teams (A and B). The x-axis represents the players, the y-axis represents the hours played, and there are two sets of bars for each player, one for each team. The legend distinguishes between the two teams.

Code:

# Importing necessary libraries
import matplotlib.pyplot as plt
import pandas as pd
import numpy as np

# Defining data for player stream A and B
playerstreamA = [
    {'name':'player1a','hours':15},
    {'name':'player2a','hours':45},
    {'name':'player3a','hours':32}
]

playerstreamB = [
    {'name':'player1b','hours':115},
    {'name':'player2b','hours':145},
    {'name':'player3b','hours':132}
]

# Creating DataFrames for player streams A and B using pandas
df_playerstreamA = pd.DataFrame(playerstreamA)
df_playerstreamB = pd.DataFrame(playerstreamB)

# Displaying the DataFrames
df_playerstreamA
df_playerstreamB

# Creating an array of evenly spaced values for the x-axis based on the length of player stream A
x = np.arange(len(df_playerstreamA))

# Creating a subplot with one plot (fig1) and one set of axes (ax1)
fig1, ax1 = plt.subplots(1, 1)

# Creating bar plots for player streams A and B on the same axes
# Bars for team A are positioned slightly to the left (shifted by -0.2) and have a width of 0.4
ax1.bar(x - 0.2, df_playerstreamA.hours, width=0.4, align='edge', label='TeamA')

# Bars for team B are positioned slightly to the right (shifted by 0.2) and have a width of 0.2
ax1.bar(x + 0.2, df_playerstreamB.hours, width=0.2, align='edge', label='teamB')

# Adding labels and title to the plot
ax1.set_xlabel('X-axis')
ax1.set_ylabel('Y-axis')
ax1.set_title('Random Sample')

# Setting x-axis ticks with player names
plt.xticks(x, df_playerstreamA.name)

# Adding a legend to the plot
plt.legend()

# Displaying the plot
plt.show()
